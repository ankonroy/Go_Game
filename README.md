# Go Game with AI Agents

A Python implementation of the classic board game Go (Weiqi/Baduk) featuring multiple AI agents using Minimax and Monte Carlo Tree Search algorithms. The game includes a graphical interface built with Pygame and supports various game modes.

## Features

- 🎮 **Full 19x19 Go board implementation**
- 🤖 **Multiple AI Agents** with different personalities:
  - **Albi**: Aggressive, capture-focused AI
  - **Ankon**: Defensive, territory-focused AI
- 🧠 **Two AI Algorithms**:
  - Minimax with Alpha-Beta pruning
  - Monte Carlo Tree Search (MCTS)
- 🎯 **Game Modes**:
  - Human vs Human
  - Human vs AI
  - AI vs AI (watch two AIs battle!)
- ⚙️ **Customizable Settings**:
  - Choose AI algorithm per player
  - Select AI personality
  - Adjust AI difficulty/depth
- 🎨 **Beautiful GUI**:
  - Traditional 19x19 Go board
  - Star points (hoshi)
  - Move highlighting
  - Coordinate display
  - Real-time score tracking
